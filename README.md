## Sobre

### Projeto de um randomizador de filmes feito usando React, Tailwind e a api "The Movie Database" para praticar o uso de Custom Hooks. 

## Como usar

### 1. Clonar o projeto em sua máquina:

```
git clone https://github.com/isamsm/movie-finder
```

### 2. Instalar dependências:

```
npm i
```

### 3. Rodar o projeto:

```
npm start
```

### 4. Acessar o ambiente local:

localhost:3000 no seu navegador.

E pronto, dessa forma você já terá o projeto rodando em sua máquina!
